# Marvellous Ajayi - Creative Developer Portfolio

A modern, responsive portfolio website showcasing the work and skills of Marvellous Ajayi, a creative developer specializing in building innovative digital experiences with cutting-edge technology and stunning design.

## 🚀 Features

- **Responsive Design**: Optimized for all devices (desktop, tablet, mobile)
- **Smooth Animations**: Engaging CSS animations and JavaScript interactions
- **Single Page Application**: Seamless navigation with smooth scrolling
- **Accessibility**: WCAG compliant with proper ARIA labels and keyboard navigation
- **Performance Optimized**: Preloaded critical assets and optimized loading
- **SEO Friendly**: Meta tags and Open Graph integration for social sharing

## 🚀 Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)
- No server required - this is a static website

### Installation

1. Clone or download the repository
2. Navigate to the project directory
3. Open `index.html` in your web browser

That's it! The website will load and display the portfolio.

## 📱 Sections

- **Home/Hero**: Introduction with animated typing effect
- **About**: Personal background and story
- **Skills**: Technical skills and expertise
- **Projects**: Featured work with GitHub links
- **CV**: Downloadable resume
- **Contact**: Contact information and social links

## 🎨 Customization

### Adding Projects

To add new projects to the portfolio:

1. Add project images to the `images/` folder
2. Edit `projects.html` to include new project cards
3. Each project should include:
   - Project title
   - Description
   - GitHub repository link
   - Technology tags
   - Optional: Featured image

### Styling

- Main styles: `css/style.css`
- Responsive design: `css/responsive.css`
- Animations: `css/animations.css`

### Content Updates

- Update personal information in `index.html`
- Modify CV content in `cv.html`
- Add/remove projects in `projects.html`
